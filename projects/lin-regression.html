Linear Regression Modelling Project

Summary: This project was inspired by a Udemy course on data science techniques for regression and forecasting. Linear regression is a cornerstone of data science and has a diverse range of applications. I learn the best through hands-on learning so I think this project was a great way to learn about regression modelling techniques and how to practice them with Python. I learned a lot about the regression modelling process such as EDA principles, feature engineering, and regularized regression techniques.
Results:
  • Fitted a multiple linear regression model for predicting rental prices with 6 predictive variables from a given dataset
  • Practiced feature engineering for more descriptive predictive variables, resulting in the creation of 5 of the 6 predictive variables used
  • Used regularized regression techniques to further improve the model's performance
You can find the Jupyter notebooks for this project here: https://github.com/jtandjung/regression-project
